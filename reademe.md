# The Digital Plate: Spicy Burger Palace
### DevOps Lab Assignment 2 & 3 - Integrated Web Project

---

## 📌 Project Overview
This project is a functional **Static Web Application** developed for the **PBA/JURY Exam-I**. It showcases a "Spicy Burger Palace" restaurant menu, demonstrating the integration of web development, backend serving with Node.js, and professional DevOps version control practices.

## 🎯 Project Requirements (Lab–2)
* **Project Title:** Spicy Burger Palace.
* **Objective:** To design and develop a basic web application using modern frameworks and prepare it for version control integration.
* **Target Users:** Local customers and food enthusiasts.
* **Key Deliverable:** 1-page Project Requirement Document and a 4-page static UI.
